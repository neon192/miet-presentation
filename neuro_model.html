Архитектура нейронной сети представляет собой гибридную модель, объединяющую преимущества сверточных нейронных сетей (CNN) для обработки пространственных данных и рекуррентных нейронных сетей (LSTM) для анализа временных последовательностей.

График 1 демонстрирует процесс обучения модели: сходимость функции потерь и рост точности предсказаний. Модель достигает стабильных результатов после 50 эпох обучения.

График 2 показывает превосходство нашей модели над традиционными методами рассеивания выбросов. Модель демонстрирует улучшение точности на 24% по сравнению с гауссовыми методами и на 15% по сравнению с Random Forest.

Технические детали

Параметры модели:

  • Входные данные: 24-часовые временные ряды + пространственная матрица 64×64
  • Скрытые слои: 3 слоя LSTM (256 нейронов) + 2 слоя CNN (128 фильтров)
  • Выход: прогноз концентрации загрязняющих веществ на 12 часов вперед
  • Оптимизатор: Adam с learning rate 0.001
  • Batch size: 32, количество эпох: 100
